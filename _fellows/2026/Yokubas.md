project_goal: >
  This project aims to extend the capabilities of the Uproot Python library by enabling modification
  of existing ROOT TTrees. The work focuses on completing the implementation of adding new branches
  (columns) to existing TTrees by finalizing and merging an existing pull request, and then designing
  and implementing the ability to append new entries (rows) to existing TTrees. The project involves
  understanding ROOT’s internal basket and metadata structure, ensuring compatibility with CERN ROOT,
  and improving support for backfilling and evolving datasets in high-energy physics workflows.
