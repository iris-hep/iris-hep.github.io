project_title: Logic Gate Neural Networks for Jet Substructure Classification
project_goal: >
  This project investigates logic neural networks (LNNs) for low-latency
  jet-substructure classification in high-energy physics. Using torchlogix, it
  will establish a reproducible workflow that trains differentiable logic
  networks, discretizes them into Boolean circuits, and validates compiled
  inference against the discrete software model. The study will evaluate
  trade-offs among classification accuracy, Boolean-circuit complexity, and
  inference latency to support hardware-efficient deployment in trigger-oriented
  workflows.
